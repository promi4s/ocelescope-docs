This page should become the reader-facing map of the documentation.

The story should be researcher-friendly: Ocelescope is both a tool for analyzing object-centric event logs and a framework for sharing process mining prototypes as runtime-loadable plugins.

Route users by intent:

- Use Ocelescope as an analyst in the User Guide.
- Build and package a plugin in Plugin Development.
- Contribute to the platform in the Developer Guide.
- Browse and submit reusable tools in Plugins.
- Find community and feedback channels in Community.
