---
title: Extensions
description: Attach plugin-specific data to OCEL logs.
---

This page should teach plugin authors how and when to implement OCEL extensions.

The story should cover extension metadata, supported file suffixes, detecting extension data, importing extension data, exporting extension data, and requiring an extension on an OCEL input.

Make the decision rule clear: use extensions for data that must travel with a log, not for ordinary result artifacts.
