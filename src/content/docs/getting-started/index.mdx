This section should guide a new user from zero to the first useful interaction with Ocelescope.

The story is practical: install Docker, start the services, open the frontend, load an OCEL log, inspect the Log Overview, upload a plugin, and run one plugin method.

Keep the conceptual explanations short here. Link to the concept pages when users need background on OCEL, object-centric process mining, plugins, or resources.
