This section should explain plugins as the main unit of functionality in Ocelescope.

The story should be: Ocelescope provides the shared workspace and runtime, while plugins provide concrete process mining functions, resources, visualizations, and integrations.

This page should help users decide whether they want to browse existing plugins, try example plugins, submit their own plugin, or learn plugin development.
