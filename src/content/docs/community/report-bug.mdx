title: Report a Bug
description: Provide actionable information for Ocelescope maintainers.
---

This page should explain how to file useful bug reports.

The story should ask for the affected version, runtime setup, browser if frontend-related, OCEL file details if import-related, plugin ZIP or source if plugin-related, reproduction steps, expected behavior, actual behavior, and logs or screenshots.

Keep the page short and link directly to the GitHub issue template.
